# Laravel-CSV-Seeder
> Seed your Laravel project's DB with CSV data. 

## Installation

[PHP](https://php.net) 5.4+ or [HHVM](http://hhvm.com) 3.3+, and [Composer](https://getcomposer.org) are required.
